## Custom Fields

| 字段 | 类型 | 建议值 |
|---|---|---|
| 优先级 | 单选 | P0 / P1 / P2 / P3 |
| 需求类型 | 单选 | 新功能 / Bug / 优化 / 技术债 / 配置 |
| 影响面 | 多选 | 前端 / 后端 / DB / 权限 / 支付 / 日志 / 报表 |
| OpenSpec Change ID | 文本 | 例如 add-login-rate-limit |
| PR 链接 | 文本 | PR URL |
| 验收人 | 人员 | 业务或产品 owner |
| 是否需要 DB 变更 | 单选 | 是 / 否 / 待确认 |
| 是否需要灰度/回滚 | 单选 | 是 / 否 / 待确认 |
| 风险等级 | 单选 | 低 / 中 / 高 |
